bootstrap弹窗-Modal插件 教程

插件介绍

  1. Bootstarp有自带的弹窗插件，Modal插件。
  2. 模态框（Modal）是覆盖在父窗体上的子窗体。通常，目的是显示来自一个单独的源的内容，可以在不离开父窗体的情况下有一些互动。子窗体可提供信息、交互等。

弹窗效果图

插件用法

  1. 引用Bootstarp插件的js。

  2. 通过 data 属性：在控制器元素（比如按钮或者链接）上设置属性 data-toggle="modal"，同时设置 data-target="# identifier " 或 href="# identifier " 来指定要切换的特定的模态框（带有 id=" identifier "）。

    通过 JavaScript：使用这种技术，您可以通过简单的一行 JavaScript 来调用带有 id="identifier" 的模态框： $('#identifier').modal(options)

产品实例

此实例来自爱康项目中数据看板的一个提交窗口。

代码讲解

  1. 使用模态窗口，您需要有某种触发器。此实例为按钮触发。
  2. 在模态框中需要注意两点：

    第一是 .modal，用来把 <div> 的内容识别为模态框。

    第二是 .fade class。当模态框被切换时，它会引起内容淡入淡出。

  3. aria-labelledby="myModalLabel"，该属性引用模态框的标题。
  4. 属性 aria-hidden="true" 用于保持模态窗口不可见，直到触发器被触发为止（比如点击在相关的按钮上）。
  5. <div class="modal-header">，modal-header 是为模态窗口的头部定义样式的类。
  6. class="close"，close 是一个 CSS class，用于为模态窗口的关闭按钮设置样式。
  7. data-dismiss="modal"，是一个自定义的 HTML5 data 属性。在这里它被用于关闭模态窗口。
  8. class="modal-body"，是 Bootstrap CSS 的一个 CSS class，用于为模态窗口的主体设置样式。
  9. class="modal-footer"，是 Bootstrap CSS 的一个 CSS class，用于为模态窗口的底部设置样式。
  10. data-toggle="modal"，HTML5 自定义的 data 属性 data-toggle 用于打开模态窗口。

注意事项

  1. 实例中未使用js环节，是因为#myModal在引用的框架js中有生成。若换成其他id，或者一个页面多个弹窗时需写入js： $('#identifier').modal(options)
  2. 公司虽然采用Bootstarp框架，但大多数的弹窗仍然采用的layui，请根据实际情况选用。
